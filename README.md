Covers VADER baseline, DistilBERT fine-tuning, Aspect-Based Sentiment Analysis,
an interactive Streamlit dashboard, and a Flask REST API.

## Step 1 вҖ” Preprocessing

```bash
python src/preprocessing.py
```

**What it does:**
- Loads the raw CSV (551k+ rows)
- Assigns sentiment labels: rating вүҘ4 вҶ’ Positive В· =3 вҶ’ Neutral В· вүӨ2 вҶ’ Negative
- Draws a stratified 50k sample preserving class proportions
- Cleans text: removes HTML tags, URLs, special characters, whitespace
- Tokenises, removes stopwords, lemmatises (NLTK)
- Splits 70% train / 15% val / 15% test (stratified)
- Saves 4 CSVs to `data/processed/`

**Output files:**
```
data/processed/full_processed.csv   (50,000 rows)
data/processed/train.csv            (35,000 rows)
data/processed/val.csv              ( 7,500 rows)
data/processed/test.csv             ( 7,500 rows)
```

**Expected time:** ~3вҖ“5 minutes on CPU

## Step 2 вҖ” VADER Sentiment Analysis

```bash
python src/vader_sentiment.py
```

**What it does:**
- Loads `data/processed/full_processed.csv`
- Runs VADER on the original `reviewText` (not cleaned text вҖ” keeps punctuation for accuracy)
- Classifies: compound вүҘ 0.05 вҶ’ Positive В· вүӨ -0.05 вҶ’ Negative В· else вҶ’ Neutral
- Evaluates accuracy, Macro F1, Precision, Recall vs star-rating ground truth
- Generates 5 plots saved to `outputs/eda_plots/`
- Saves `outputs/vader_results.csv`

**Output files:**
```
outputs/vader_results.csv
outputs/eda_plots/vader_01_compound_distribution.png
outputs/eda_plots/vader_02_confusion_matrix.png
outputs/eda_plots/vader_03_scores_per_category.png
outputs/eda_plots/vader_04_label_dist_per_category.png
outputs/eda_plots/vader_05_performance_summary.png
```

**Expected accuracy:** ~75вҖ“80%
**Expected time:** ~2вҖ“3 minutes on CPU

## Step 3 вҖ” BERT Fine-Tuning

```bash
python src/bert_model.py
```

**What it does:**
- Loads `data/processed/train.csv`, `val.csv`, `test.csv`
- Fine-tunes `distilbert-base-uncased` for 3-class sentiment classification
- Uses HuggingFace Trainer API with:
  - AdamW optimiser (lr = 2e-5)
  - Linear warmup (10% of steps)
  - Weight decay = 0.01
  - EarlyStopping patience = 2 epochs
- Evaluates on test set: Accuracy В· Macro F1 В· Precision В· Recall В· Confusion Matrix
- Compares BERT vs VADER vs TF-IDF + Logistic Regression baseline
- Saves best checkpoint with `model.save_pretrained()`

**Output files:**
```
models/bert_checkpoint/pytorch_model.bin   (upload to Drive if >100MB)
models/bert_checkpoint/config.json
models/bert_checkpoint/tokenizer files
outputs/bert_results.csv
outputs/eda_plots/bert_01_confusion_matrix.png
outputs/eda_plots/bert_02_training_history.png
outputs/eda_plots/bert_03_model_comparison.png
```

**Expected accuracy:** ~88вҖ“93%
**Expected time:** ~30 min on GPU В· ~90 min/epoch on CPU

## Step 4 вҖ” Aspect-Based Sentiment Analysis (ABSA)

```bash
python src/absa.py
```

**What it does:**
- Loads `data/processed/full_processed.csv`
- Extracts product aspects using SpaCy dependency parsing + keyword matching
- Detects 6 aspects: **Quality В· Price В· Delivery В· Packaging В· Customer Service В· Design**
- Scores VADER sentiment on the relevant sentences per aspect (not the full review)
- Aggregates mean compound score per category Г— aspect вҶ’ scorecard
- Generates 4 plots including the required heatmap

**Output files:**
```
outputs/absa_scorecard.csv              (categories Г— aspects mean scores)
outputs/absa_review_details.csv         (per-review aspect scores)
outputs/eda_plots/absa_01_heatmap.png
outputs/eda_plots/absa_02_aspect_distribution.png
outputs/eda_plots/absa_03_radar.png
outputs/eda_plots/absa_04_boxplots.png
```

**Expected time:** ~10вҖ“15 minutes on CPU (SpaCy parses 50k reviews)
**Progress:** prints every 5,000 reviews

## Running the Flask REST API

```bash

```

API runs at: **http://localhost:5000**

**Required endpoints (assignment spec):**

```bash

curl http://localhost:5000/health

# POST /analyse вҖ” single review: VADER + BERT + aspects

     -H "Content-Type: application/json" \
     -d "{\"text\": \"Great quality product, fast delivery!\"}"
```
